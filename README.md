AI小说写作软件
一个基于大模型API的智能小说写作工具，支持单章写作、批量写作、自动写完整本，帮你轻松创作百万字长篇小说。

✨ 功能特性
📝 多种写作模式：单章写作、批量多章、自动写完整本

🎯 智能大纲生成：根据小说描述自动生成详细大纲

👤 角色自动创建：基于大纲生成一致的角色设定

🌍 世界观构建：自动生成完整的世界观设定

🤖 自动标题生成：每章自动生成吸引人的标题

📄 自动传递上下文：写作时自动总结前情提要

🔍 逻辑一致性检查：检查角色名、时间线、伏笔是否一致

✨ 文本润色：自动优化文笔，增强画面感

💾 自动保存：每5章自动保存，支持断点续传

⏹ 随时停止：批量写作中可随时停止

📤 导出TXT：一键导出整本小说

🚀 快速开始
安装依赖
bash
pip install -r requirements.txt
运行程序
bash
python main.py
使用步骤
配置AI模型：点击工具栏「⚙ 配置AI」，填写API信息

创建项目：填写小说名称、类型、风格、描述

生成大纲：点击「生成大纲」自动规划故事框架

创建角色：基于大纲生成角色（确保人名一致）

生成世界观：自动构建世界观设定

开始写作：选择写作模式，开始创作

📖 使用说明
写作模式
单章写作：手动控制每一章，可指定大纲和特殊要求

批量写作：指定章节范围，连续自动写作

自动写完整本：输入总章节数，一键写完整本小说

写作建议
描述要详细：创建项目时的描述越详细，AI理解越准确

先测试一章：批量写作前先单章测试，确认风格满意

定期检查：每10章左右检查一次内容质量

善用润色：对不满意的章节可以单独润色

及时保存：虽然会自动保存，但建议手动备份

避免AI味道的技巧
在特殊要求中注明「避免解说性文字」

使用润色功能优化平淡段落

手动修改AI常见的重复句式

保持角色性格一致性

🐛 常见问题
Q: 提示「API返回格式异常」？
A: 检查API地址是否正确，确保模型名称无误。

Q: 大纲生成很慢？
A: 正常需要20-60秒，请耐心等待。如超过2分钟，检查网络。

Q: 角色名和大纲不一致？
A: 请先生成大纲，再点击创建角色，角色会基于大纲生成。

Q: 写作时卡顿？
A: 批量写作在后台线程运行，界面不会卡顿。如感觉慢，是API响应时间。

Q: 停止后闪退？
A: 程序会等待当前章节写完再停止，请稍等片刻。

Q: 如何使用免费模型？
A: 安装Ollama，拉取模型后在软件中选择「Ollama本地」。
